# GTS: Govind Tiles & Sanitary - Intelligent Inventory Dashboard 🧱

**GTS** is a professional-grade Data Science application designed to transform raw inventory data into actionable business insights. Built specifically for the tiles and sanitaryware industry, this tool helps in managing complex stock levels, identifying trends, and optimizing warehouse space.

### 📊 Advanced Analytics

### 🛠️ Operational Tools
* [cite_start]**Smart Inventory Explorer**: A fast search engine to locate specific tile codes or designs instantly[cite: 89, 90].
* [cite_start]**Low Stock Alerts**: Automatic flagging of items with critical stock levels (Quantity < 10)[cite: 96].
